Rules:

The objective of Simon Says is to follow along with the computer as they select random colors in the play area. Every game will start with the computer selecting a color, the user must then follow along selecting the same exact colors in the same exact order.

Losing the game:

If the user misses a press or presses the wrong color, it is a game over. At that point the user myust hit the reset button and then start again to play the next game of Simon Says.

Winning the game:

If the user can successfully complete 20 rounds the game will end and the user will declared the winner.
